Sales Data Analysis Project
🔍 Overview

This project analyzes sales data to understand how price, discount, and category affect revenue.

📁 Dataset
5000 rows , 8 features
Includes missing values (cleaned during analysis)
🧹 Data Cleaning
Handled missing values using mean/median/mode
Removed duplicates
Converted categorical variables
📊 EDA (Exploratory Data Analysis)
Sales distribution
Category vs Sales
Correlation heatmap
Discount impact analysis

📈 Key Insights
Higher discounts increase sales
Some categories perform better
Price has mixed impact
